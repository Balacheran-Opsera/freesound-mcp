{% comment %}This is a custom version of the standard navbar which by default hides the input box (by adding the "bw-nav--expanded" class to the navbar from a start). This version also sets the "value" parameter of the input element so that if there is a query defined, the input box will be pre-filled with that. Furthermore, this version of the navbar has an ID defined for the form and input elements so we can intercept the submit event and do some actions to include advance search options (and other fields) before submission.{% endcomment %}
